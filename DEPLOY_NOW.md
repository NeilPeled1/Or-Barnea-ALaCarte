# Deploy – One Click

**Click to deploy:** https://vercel.com/new/import?url=https://github.com/NeilPeled1/Or-Barnea-ALaCarte&project-name=Or-Barnea-ALaCarte

1. Sign in with GitHub
2. Add env vars: `AUTH_SECRET` (run `openssl rand -base64 32`), `NEXTAUTH_URL` = `https://Or-Barnea-ALaCarte.vercel.app`
3. Deploy (database optional – demo login works without it)

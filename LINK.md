# À La Carte – Share This Link

## Deploy (one-time setup)

**Click:** https://vercel.com/new/import?url=https://github.com/NeilPeled1/or-barnea-alacarte&project-name=or-barnea-alacarte

1. Sign in with GitHub
2. Add Environment Variables:
   - `AUTH_SECRET` = run `openssl rand -base64 32` in terminal, paste result
   - `NEXTAUTH_URL` = `https://or-barnea-alacarte.vercel.app`
3. Click Deploy
